# Sou Saúde - Firefox OS App

## Descrição
App criado para a Hackathon de Firefox OS na campus party.
O App auxilia no seus treinos diários. Controlar os seus treinos podem ser uma tarefa dificil, este app lhe ajuda a manter os seus treinos organizados e nunca mais esquecer quais exercicios você deveria fazer.

## Vantagens
* Fácil de usar
* Não precisa gastar papel
* Não precisa ficar decorando quais exercicios devem ser executados em cada treino, o App lhe diz
* Roda no Firefox OS
* É livre, use, modifique, compartilhe
